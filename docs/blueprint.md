# **App Name**: Britke Sablje Landing

## Core Features:

- Hero Section: Displays club name, subheadline, and a prominent call-to-action button for scheduling a trial training session.
- Floating Dock Navigation: Fixed navigation bar with links to 'Poƒçetna', 'O Klubu', 'Tehnike', and 'Kontakt', offering a seamless user experience.
- Technique Showcase: Interactive grid showcasing arm wrestling styles: 'Kuka (Hook)', 'Top Roll', and 'Pres (Press)', each with a brief explanation. Hover effect is implemented by the tool which helps users find specific technniques with ease.

## Style Guidelines:

- Background: Deep Royal Purple to Black gradient (from #0F0518 to #000000) to create a premium sport dark mode feel.
- Primary Accent: Metallic Silver for borders and subtle highlights, adding a touch of sophistication.
- Action Color: Burnt Orange (#F97316) exclusively for buttons and active states, drawing user attention to interactive elements.
- Body text: 'Inter' (sans-serif) for a clean and readable appearance.
- Headings: 'Russo One' (sans-serif) to reflect strength and stability.
- Minimalist layout emphasizing negative space to enhance readability and create a sophisticated, clean look.
- Framer Motion is used for physics-based 3D Tilt effect on Technique Showcase cards, adding a layer of interactivity and depth.
